# Ironhold-fps-demo
First Person Shooter demo using Ironhold

## Online demo
The online demo gets automatically updated when changes are pushed to main.  
**Play it live** https://frankvangeest.github.io//Ironhold-fps-demo/index.html?project=scifi_fps

## Ironhold Engine
This project is made using the Ironhold engine.  
See https://github.com/frankvangeest/ironhold-lib for more info.

### Limitations
All scene setup and behaviours are setup using the ron files.  
No compilation needed. We use the latest wasm web build of the engine as is.  
This demo is limited to the available features the engine provides.  
If you want to see more features in action, see examples https://frankvangeest.github.io/ironhold-lib/index.html
